# Política de segurança

Este repositório contém uma prova de conceito educacional.

## Não publicar

- diretório `output/tokens/`;
- chaves privadas ou backups de tokens;
- arquivo `.env` de ambientes reais;
- PINs ou credenciais de produção;
- payloads ou dados reais de medicamentos;
- certificados privados ou segredos de integração.

## Vulnerabilidades

Não publique segredos ou vulnerabilidades sensíveis em uma issue pública. Entre em contato com a autora por um canal privado disponível no perfil do GitHub e informe apenas o necessário para reprodução segura.
